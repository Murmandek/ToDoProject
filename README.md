# ToDoProject
ToDo list is a simple web application to save employee tasks in order not to miss anything.

### Features
* Add new task
* Edit task
* Delete single task
* Delete all tasks
* Add new employee with avatar
* Edit employee
* Delete single employee
* Flexible search of employees and tasks

### Technologies used:
* ASP.NET Core
* Entity Framework
* HTML
* CSS
* JavaScript/jQuery 
* SQL
